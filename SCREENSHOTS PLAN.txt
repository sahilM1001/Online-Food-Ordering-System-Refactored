STEPS TO RUN PROJECT:
----------------------------------------------------
Open XAMPP, run mySQL admin, go to onlinefooddeliverysystem Database, then go to import.

Now import SQL file onlinefooddeliverysystem FINAL.sql from DB folder in project.

New Database should be loaded

go to project folder and run project using python manage.py runserver
